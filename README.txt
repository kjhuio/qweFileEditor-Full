qweファイルエディターについて
本qweファイルエディター(以下エディター)は、qwe0412(github:kjhuio)が作成したテキストエディターです。
本エディターは、独自拡張子(*.qef)やHTML、CSS、その他ほとんどのファイルに対応しています。
まだ扱えるファイルは、プレーンテキストのみですが、これからいろいろなファイルに対応していく予定です。
